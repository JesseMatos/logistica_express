# logistica_express
Uma empresa de transporte rodoviario ficticia necessitando de ajuda em algumas de suas operações.

Neste projeto usamos as seguintes aplicações:
- **Modularização:** criação de funções específicas para cada cálculo.
- **Passagem de Parâmetros:** envio de dados (distância, tempo, velocidade) para as funções.
- **Retorno de Valores:** uso da instrução 'retorne' para devolver os resultados ao programa principal.
- **Interação com o Usuário:** entrada e saída de dados via console no Porgugol Studio.

**Etapa 1 - Cálculo de Tempo**
- Criação de uma função específica para o usuário calcular o **cronômetro da carga** de determinado item com base em parâmetros determinados.

**Etapa 2 - Cálculo de Velocidade Média**
- Criação de uma função específica para o usuário calcular a **meta de entrega** dee determinado item para chegada em seu destino final no tempo estabelecido.

**Etapa 3 - Cálculo de Distância**
- Criação de uma função específica para o usuário calcular **autonomia de rota** que determinado item deve percorrer com base em parâmetros determinados.
